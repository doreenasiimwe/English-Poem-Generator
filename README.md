# 🎵 AI English Poem Generator

This is a simple AI-powered web application that generates short English poems based on user input.

## ✨ Features
- Users can enter a topic (e.g. love, rain, friendship)
- AI generates a 4-line poem
- Clean and responsive UI
- Animated poem display using Typewriter effect

## 🛠️ Built With
- HTML
- CSS
- JavaScript
- Axios
- SheCodes AI API
- Typewriter.js

## 🚀 How It Works
1. User enters a topic into the input field
2. The app sends the prompt to an AI API
3. AI returns a 4-line poem
4. The poem is displayed with animation
